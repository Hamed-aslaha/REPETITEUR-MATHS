Équations du Second Degré

Résoudre une équation du second degré de la forme ax² + bx + c = 0 en utilisant le discriminant.

Formule du discriminant

Δ = b² - 4ac

  • Si Δ > 0, l'équation admet deux solutions réelles distinctes
  • Si Δ = 0, l'équation admet une solution réelle double
  • Si Δ < 0, l'équation n'admet pas de solution réelle

Exemple 1

Résoudre l'équation : 2x² - 5x + 2 = 0

Étape 1: Identifions les coefficients

a = 2, b = -5, c = 2

Étape 2: Calculons le discriminant

Δ = b² - 4ac = (-5)² - 4 × 2 × 2 = 25 - 16 = 9

Étape 3: Comme Δ > 0, l'équation admet deux solutions réelles distinctes

x₁ = (-b - √Δ) / 2a = (5 - 3) / 4 = 2/4 = 0,5

x₂ = (-b + √Δ) / 2a = (5 + 3) / 4 = 8/4 = 2

Les solutions sont donc x₁ = 0,5 et x₂ = 2

Exemple 2

Résoudre l'équation : 3x² + 6x + 3 = 0

Étape 1: Identifions les coefficients

a = 3, b = 6, c = 3

Étape 2: Calculons le discriminant

Δ = b² - 4ac = 6² - 4 × 3 × 3 = 36 - 36 = 0

Étape 3: Comme Δ = 0, l'équation admet une solution réelle double

x = -b / 2a = -6 / (2 × 3) = -6/6 = -1

La solution est donc x = -1 (solution double)

On peut aussi factoriser : 3x² + 6x + 3 = 3(x² + 2x + 1) = 3(x + 1)²

Exercices à faire

Exercice 1: Résoudre les équations suivantes

  1. x² - 4x + 4 = 0
  2. 2x² + 5x - 3 = 0
  3. -x² + 2x - 1 = 0

1. x² - 4x + 4 = 0

a = 1, b = -4, c = 4

Δ = b² - 4ac = (-4)² - 4 × 1 × 4 = 16 - 16 = 0

Δ = 0, donc une solution double : x = -b/(2a) = 4/2 = 2

Solution : x = 2 (double)

2. 2x² + 5x - 3 = 0

a = 2, b = 5, c = -3

Δ = b² - 4ac = 5² - 4 × 2 × (-3) = 25 + 24 = 49

Δ > 0, donc deux solutions réelles :

x₁ = (-b - √Δ)/(2a) = (-5 - 7)/(2×2) = -12/4 = -3

x₂ = (-b + √Δ)/(2a) = (-5 + 7)/(2×2) = 2/4 = 0,5

Solutions : x₁ = -3 et x₂ = 0,5

3. -x² + 2x - 1 = 0

a = -1, b = 2, c = -1

Δ = b² - 4ac = 2² - 4 × (-1) × (-1) = 4 - 4 = 0

Δ = 0, donc une solution double : x = -b/(2a) = -2/(-2) = 1

Solution : x = 1 (double)

Exercice 2: Factoriser les expressions suivantes

  1. x² - 9
  2. 4x² - 12x + 9
  3. x² + 6x + 9

1. x² - 9

C'est une différence de carrés : a² - b² = (a+b)(a-b)

x² - 9 = x² - 3² = (x+3)(x-3)

2. 4x² - 12x + 9

On peut factoriser par 1 : 4x² - 12x + 9 = 4(x² - 3x) + 9

Mais on peut aussi reconnaître un carré parfait :

4x² - 12x + 9 = (2x)² - 2(2x)(3) + 3²

= (2x - 3)²

3. x² + 6x + 9

On reconnaît un carré parfait : x² + 6x + 9 = x² + 2(x)(3) + 3²

= (x + 3)²

Limites et Continuité

Règles de calcul des limites

  • lim(x→a) [f(x) + g(x)] = lim(x→a) f(x) + lim(x→a) g(x)
  • lim(x→a) [f(x) × g(x)] = lim(x→a) f(x) × lim(x→a) g(x)
  • lim(x→a) [f(x) / g(x)] = lim(x→a) f(x) / lim(x→a) g(x), si lim(x→a) g(x) ≠ 0
  • lim(x→+∞) (1/x) = 0
  • lim(x→+∞) (x^n) = +∞ pour n > 0

Exemple 1

Calculer la limite de f(x) = (3x² - 2x + 1) / (x² + 4) quand x tend vers +∞

Méthode: Pour calculer la limite d'une fraction rationnelle quand x tend vers l'infini, on divise le numérateur et le dénominateur par la plus grande puissance de x.

f(x) = (3x² - 2x + 1) / (x² + 4)

Divisons par x² (la plus grande puissance) :

f(x) = (3 - 2/x + 1/x²) / (1 + 4/x²)

Quand x tend vers +∞ :

lim(x→+∞) (2/x) = 0

lim(x→+∞) (1/x²) = 0

lim(x→+∞) (4/x²) = 0

Donc :

lim(x→+∞) f(x) = (3 - 0 + 0) / (1 + 0) = 3/1 = 3

La limite est 3

Exemple 2

Étudier la continuité de la fonction f définie par :

f(x) = { (x² - 4) / (x - 2) si x ≠ 2

{ 4 si x = 2

Méthode: Pour étudier la continuité en x = 2, nous devons vérifier si la limite de f(x) quand x tend vers 2 est égale à f(2).

Calculons la limite de f(x) quand x tend vers 2 :

lim(x→2) f(x) = lim(x→2) (x² - 4) / (x - 2)

= lim(x→2) ((x - 2)(x + 2)) / (x - 2)

= lim(x→2) (x + 2) = 2 + 2 = 4

Or, f(2) = 4 d'après la définition de la fonction.

Donc, lim(x→2) f(x) = f(2) = 4

La fonction est continue en x = 2.

Exercice 1: Calculer les limites suivantes

  1. lim(x→+∞) (2x³ + x) / (x³ - 5x² + 1)
  2. lim(x→0) (sin(x)) / x
  3. lim(x→1) (x² - 1) / (x - 1)

1. lim(x→+∞) (2x³ + x) / (x³ - 5x² + 1)

Divisons par x³ (plus haute puissance) :

(2x³ + x) / (x³ - 5x² + 1) = (2 + 1/x²) / (1 - 5/x + 1/x³)

Quand x→+∞, 1/x², 5/x et 1/x³ tendent vers 0

Donc lim(x→+∞) (2x³ + x) / (x³ - 5x² + 1) = 2/1 = 2

2. lim(x→0) (sin(x)) / x

C'est une limite fondamentale : lim(x→0) (sin(x)) / x = 1

3. lim(x→1) (x² - 1) / (x - 1)

Factorisons le numérateur :

(x² - 1) / (x - 1) = ((x - 1)(x + 1)) / (x - 1) = x + 1

Donc lim(x→1) (x² - 1) / (x - 1) = 1 + 1 = 2

Comme lim(x→0⁻) g(x) ≠ lim(x→0⁺) g(x), la fonction g n'est pas continue en x = 0

La fonction g est continue sur ]-∞, 0[ et sur ]0, +∞[

Géométrie dans l'espace

Vecteurs dans l'espace

Un vecteur dans l'espace est caractérisé par ses trois coordonnées dans un repère orthonormé (O, i, j, k).

Si A(x₁, y₁, z₁) et B(x₂, y₂, z₂) sont deux points de l'espace, alors :

AB = (x₂ - x₁, y₂ - y₁, z₂ - z₁)

Le produit scalaire de deux vecteurs u(x₁, y₁, z₁) et v(x₂, y₂, z₂) est :

u·v = x₁x₂ + y₁y₂ + z₁z₂

Exemple 1

Soient les points A(1, 2, 3), B(4, 0, 2) et C(2, 1, 5). Calculer le produit scalaire AB·AC.

Étape 1: Calculons les coordonnées des vecteurs AB et AC

AB = (4-1, 0-2, 2-3) = (3, -2, -1)

AC = (2-1, 1-2, 5-3) = (1, -1, 2)

Étape 2: Calculons le produit scalaire AB·AC

AB·AC = 3×1 + (-2)×(-1) + (-1)×2

= 3 + 2 - 2 = 3

Le produit scalaire AB·AC = 3

Exemple 2

Déterminer si les vecteurs u(2, 1, -3) et v(-4, -2, 6) sont colinéaires.

Méthode: Deux vecteurs u(x₁, y₁, z₁) et v(x₂, y₂, z₂) sont colinéaires s'il existe un réel k tel que v = k×u, c'est-à-dire si x₂ = k×x₁, y₂ = k×y₁ et z₂ = k×z₁.

Vérifions s'il existe un réel k tel que v = k×u :

-4 = k×2 ⟹ k = -2

-2 = k×1 ⟹ k = -2

6 = k×(-3) ⟹ k = -2

On trouve la même valeur k = -2 pour chaque coordonnée, donc v = -2×u.

Les vecteurs u et v sont colinéaires.

Exercice 1: Produit scalaire et orthogonalité

Soient les vecteurs u(3, -1, 2), v(1, 2, 4) et w(0, 7, -2).

  1. Calculer u·v et u·w
  2. Les vecteurs u et v sont-ils orthogonaux ?
  3. Les vecteurs u et w sont-ils orthogonaux ?

1. Calcul des produits scalaires

u·v = 3×1 + (-1)×2 + 2×4 = 3 - 2 + 8 = 9

u·w = 3×0 + (-1)×7 + 2×(-2) = 0 - 7 - 4 = -11

2. Orthogonalité de u et v

Deux vecteurs sont orthogonaux si et seulement si leur produit scalaire est nul.

u·v = 9 ≠ 0, donc les vecteurs u et v ne sont pas orthogonaux.

3. Orthogonalité de u et w

u·w = -11 ≠ 0, donc les vecteurs u et w ne sont pas orthogonaux.

Exercice 2: Équation d'un plan

Soit un plan P passant par le point A(1, 2, -1) et de vecteur normal n(2, 3, 1).

  1. Déterminer une équation cartésienne du plan P.
  2. Le point B(3, -1, 4) appartient-il au plan P ?

1. Équation cartésienne du plan

L'équation cartésienne d'un plan de vecteur normal n(a, b, c) passant par un point A(x₀, y₀, z₀) est :

a(x - x₀) + b(y - y₀) + c(z - z₀) = 0

Avec n(2, 3, 1) et A(1, 2, -1) :

2(x - 1) + 3(y - 2) + 1(z - (-1)) = 0

2x - 2 + 3y - 6 + z + 1 = 0

2x + 3y + z - 7 = 0

2. Appartenance du point B au plan

Pour vérifier si B(3, -1, 4) appartient au plan, on remplace les coordonnées dans l'équation :

2×3 + 3×(-1) + 1×4 - 7 = 0

6 - 3 + 4 - 7 = 0

0 = 0

L'égalité est vérifiée, donc le point B appartient au plan P.

Probabilités

Formules importantes

  • P(A∪B) = P(A) + P(B) - P(A∩B)
  • P(A∩B) = P(A) × P(B) si A et B sont indépendants
  • P(A|B) = P(A∩B) / P(B) (probabilité conditionnelle)
  • Formule des probabilités totales : P(A) = P(A|B) × P(B) + P(A|B̄) × P(B̄)
  • Formule de Bayes : P(B|A) = [P(A|B) × P(B)] / P(A)

Exemple 1

Une urne contient 5 boules rouges et 3 boules bleues. On tire successivement et sans remise 2 boules de l'urne. Quelle est la probabilité d'obtenir 2 boules de la même couleur ?

Méthode: On calcule la probabilité d'obtenir 2 boules rouges ou 2 boules bleues.

Nombre total de boules : 5 + 3 = 8

Probabilité d'obtenir 2 boules rouges :

P(RR) = (5/8) × (4/7) = 20/56 = 5/14

Probabilité d'obtenir 2 boules bleues :

P(BB) = (3/8) × (2/7) = 6/56 = 3/28

Probabilité d'obtenir 2 boules de la même couleur :

P(RR ou BB) = P(RR) + P(BB) = 5/14 + 3/28 = 10/28 + 3/28 = 13/28

La probabilité est 13/28 ≈ 0,464

Exemple 2

Dans une classe, 60% des élèves sont des filles. Parmi les filles, 80% ont réussi l'examen, tandis que parmi les garçons, 70% ont réussi. On choisit un élève au hasard qui a réussi l'examen. Quelle est la probabilité que ce soit une fille ?

Méthode: On utilise la formule de Bayes.

Notons :

F : l'élève est une fille

G : l'élève est un garçon

R : l'élève a réussi l'examen

Données :

P(F) = 0,6 et P(G) = 0,4

P(R|F) = 0,8

P(R|G) = 0,7

On cherche P(F|R), la probabilité que l'élève soit une fille sachant qu'il a réussi.

D'après la formule de Bayes :

P(F|R) = [P(R|F) × P(F)] / P(R)

Calculons P(R) avec la formule des probabilités totales :

P(R) = P(R|F) × P(F) + P(R|G) × P(G)

P(R) = 0,8 × 0,6 + 0,7 × 0,4 = 0,48 + 0,28 = 0,76

Donc :

P(F|R) = (0,8 × 0,6) / 0,76 = 0,48 / 0,76 = 0,632

La probabilité est environ 0,63 (ou 63%)

Exercices à faire

Exercice 1: Probabilités conditionnelles

Une entreprise fabrique des pièces dans trois ateliers A, B et C. L'atelier A produit 20% des pièces, l'atelier B en produit 30% et l'atelier C en produit 50%. Les taux de pièces défectueuses sont respectivement de 5% pour l'atelier A, 4% pour l'atelier B et 2% pour l'atelier C.

  1. Quelle est la probabilité qu'une pièce choisie au hasard soit défectueuse ?
  2. Si une pièce est défectueuse, quelle est la probabilité qu'elle provienne de l'atelier C ?

1. Probabilité qu'une pièce soit défectueuse

Notons D l'événement "la pièce est défectueuse"

D'après la formule des probabilités totales :

P(D) = P(D|A)×P(A) + P(D|B)×P(B) + P(D|C)×P(C)

P(D) = 0,05×0,2 + 0,04×0,3 + 0,02×0,5

P(D) = 0,01 + 0,012 + 0,01 = 0,032

La probabilité qu'une pièce soit défectueuse est de 0,032 (ou 3,2%)

2. Probabilité que la pièce défectueuse provienne de l'atelier C

On cherche P(C|D)

D'après la formule de Bayes :

P(C|D) = [P(D|C)×P(C)] / P(D)

P(C|D) = (0,02×0,5) / 0,032

P(C|D) = 0,01 / 0,032 = 0,3125

La probabilité que la pièce défectueuse provienne de l'atelier C est d'environ 0,31 (ou 31%)

Exercice 2: Variable aléatoire

On lance un dé équilibré à 6 faces. On définit la variable aléatoire X qui associe à chaque résultat le carré de la différence entre le résultat et 3,5.

  1. Déterminer la loi de probabilité de X.
  2. Calculer l'espérance mathématique E(X).

1. Loi de probabilité de X

X = (résultat - 3,5)²

Pour chaque résultat du dé, calculons la valeur de X :

Si résultat = 1 : X = (1 - 3,5)² = (-2,5)² = 6,25

Si résultat = 2 : X = (2 - 3,5)² = (-1,5)² = 2,25

Si résultat = 3 : X = (3 - 3,5)² = (-0,5)² = 0,25

Si résultat = 4 : X = (4 - 3,5)² = (0,5)² = 0,25

Si résultat = 5 : X = (5 - 3,5)² = (1,5)² = 2,25

Si résultat = 6 : X = (6 - 3,5)² = (2,5)² = 6,25

La loi de probabilité de X est donc :

P(X = 0,25) = 2/6 = 1/3

P(X = 2,25) = 2/6 = 1/3

P(X = 6,25) = 2/6 = 1/3

2. Espérance mathématique E(X)

E(X) = Σ [x_i × P(X = x_i)]

E(X) = 0,25 × (1/3) + 2,25 × (1/3) + 6,25 × (1/3)

E(X) = (0,25 + 2,25 + 6,25) / 3

E(X) = 8,75 / 3 ≈ 2,92

1. Le cercle trigonométrique

Le cercle trigonométrique est un cercle de rayon 1 centré à l'origine du repère orthonormé (O, \(\vec{i}\), \(\vec{j}\)). Il permet de définir les fonctions trigonométriques.

Définitions

Pour un angle \(\theta\) dans le cercle trigonométrique :

  • Le cosinus de \(\theta\) est l'abscisse du point M : \(\cos(\theta) = x_M\)
  • Le sinus de \(\theta\) est l'ordonnée du point M : \(\sin(\theta) = y_M\)
  • La tangente de \(\theta\) est le rapport : \(\tan(\theta) = \frac{\sin(\theta)}{\cos(\theta)}\) (définie si \(\cos(\theta) \neq 0\))

Propriétés fondamentales

  • Relation fondamentale : \(\sin^2(\theta) + \cos^2(\theta) = 1\)
  • Périodicité :
    • \(\sin(\theta + 2\pi) = \sin(\theta)\)
    • \(\cos(\theta + 2\pi) = \cos(\theta)\)
    • \(\tan(\theta + \pi) = \tan(\theta)\)
  • Parité :
    • \(\sin(-\theta) = -\sin(\theta)\) (fonction impaire)
    • \(\cos(-\theta) = \cos(\theta)\) (fonction paire)
    • \(\tan(-\theta) = -\tan(\theta)\) (fonction impaire)

2. Les angles remarquables

Angle \(\theta\) 0 \(\frac{\pi}{6}\) ou \(30°\) \(\frac{\pi}{4}\) ou \(45°\) \(\frac{\pi}{3}\) ou \(60°\) \(\frac{\pi}{2}\) ou \(90°\) \(\pi\) ou \(180°\)
\(\sin(\theta)\) 0 \(\frac{1}{2}\) \(\frac{\sqrt{2}}{2}\) \(\frac{\sqrt{3}}{2}\) 1 0
\(\cos(\theta)\) 1 \(\frac{\sqrt{3}}{2}\) \(\frac{\sqrt{2}}{2}\) \(\frac{1}{2}\) 0 -1
\(\tan(\theta)\) 0 \(\frac{1}{\sqrt{3}}\) 1 \(\sqrt{3}\) Non défini 0

3. Formules trigonométriques

Formules d'addition

  • \(\sin(a + b) = \sin(a)\cos(b) + \cos(a)\sin(b)\)
  • \(\sin(a - b) = \sin(a)\cos(b) - \cos(a)\sin(b)\)
  • \(\cos(a + b) = \cos(a)\cos(b) - \sin(a)\sin(b)\)
  • \(\cos(a - b) = \cos(a)\cos(b) + \sin(a)\sin(b)\)
  • \(\tan(a + b) = \frac{\tan(a) + \tan(b)}{1 - \tan(a)\tan(b)}\)
  • \(\tan(a - b) = \frac{\tan(a) - \tan(b)}{1 + \tan(a)\tan(b)}\)

Formules de duplication

  • \(\sin(2a) = 2\sin(a)\cos(a)\)
  • \(\cos(2a) = \cos^2(a) - \sin^2(a) = 2\cos^2(a) - 1 = 1 - 2\sin^2(a)\)
  • \(\tan(2a) = \frac{2\tan(a)}{1 - \tan^2(a)}\)

Formules de linéarisation

  • \(\sin(a)\sin(b) = \frac{1}{2}[\cos(a-b) - \cos(a+b)]\)
  • \(\sin(a)\cos(b) = \frac{1}{2}[\sin(a+b) + \sin(a-b)]\)
  • \(\cos(a)\cos(b) = \frac{1}{2}[\cos(a+b) + \cos(a-b)]\)

Formules de factorisation

  • \(\sin(a) + \sin(b) = 2\sin(\frac{a+b}{2})\cos(\frac{a-b}{2})\)
  • \(\sin(a) - \sin(b) = 2\sin(\frac{a-b}{2})\cos(\frac{a+b}{2})\)
  • \(\cos(a) + \cos(b) = 2\cos(\frac{a+b}{2})\cos(\frac{a-b}{2})\)
  • \(\cos(a) - \cos(b) = -2\sin(\frac{a+b}{2})\sin(\frac{a-b}{2})\)

4. Équations trigonométriques

Équations de base

1. Équation \(\sin(x) = a\)

  • Si \(|a| > 1\), pas de solution
  • Si \(|a| \leq 1\), alors \(x = \arcsin(a) + 2k\pi\) ou \(x = \pi - \arcsin(a) + 2k\pi\), avec \(k \in \mathbb{Z}\)

2. Équation \(\cos(x) = a\)

  • Si \(|a| > 1\), pas de solution
  • Si \(|a| \leq 1\), alors \(x = \arccos(a) + 2k\pi\) ou \(x = -\arccos(a) + 2k\pi\), avec \(k \in \mathbb{Z}\)

3. Équation \(\tan(x) = a\)

  • Pour tout \(a \in \mathbb{R}\), \(x = \arctan(a) + k\pi\), avec \(k \in \mathbb{Z}\)

Exemple résolu : \(\sin(2x) = \frac{1}{2}\) sur \([0, 2\pi]\)

Solution :

On cherche les valeurs de \(x \in [0, 2\pi]\) telles que \(\sin(2x) = \frac{1}{2}\)

On sait que \(\sin(\theta) = \frac{1}{2}\) pour \(\theta = \frac{\pi}{6} + 2k\pi\) ou \(\theta = \frac{5\pi}{6} + 2k\pi\), avec \(k \in \mathbb{Z}\)

Donc \(2x = \frac{\pi}{6} + 2k\pi\) ou \(2x = \frac{5\pi}{6} + 2k\pi\)

Ce qui donne \(x = \frac{\pi}{12} + k\pi\) ou \(x = \frac{5\pi}{12} + k\pi\)

Pour \(x \in [0, 2\pi]\), on obtient les solutions :

\(x = \frac{\pi}{12}, \frac{5\pi}{12}, \frac{\pi}{12} + \pi = \frac{13\pi}{12}, \frac{5\pi}{12} + \pi = \frac{17\pi}{12}\)

5. Exercices corrigés

Exercice 1 : Simplification d'expressions

Simplifier l'expression : \(E = \sin^2(x) - \sin^2(y) - \cos^2(x) + \cos^2(y)\)

Solution :

\(E = \sin^2(x) - \sin^2(y) - \cos^2(x) + \cos^2(y)\)

On sait que \(\sin^2(t) + \cos^2(t) = 1\) pour tout réel \(t\)

Donc \(\sin^2(x) - \cos^2(x) = \sin^2(x) - (1 - \sin^2(x)) = 2\sin^2(x) - 1\)

Et \(-\sin^2(y) + \cos^2(y) = -\sin^2(y) + (1 - \sin^2(y)) = 1 - 2\sin^2(y)\)

\(E = (2\sin^2(x) - 1) + (1 - 2\sin^2(y)) = 2\sin^2(x) - 2\sin^2(y)\)

\(E = 2(\sin^2(x) - \sin^2(y))\)

\(E = 2((\sin(x) - \sin(y))(\sin(x) + \sin(y)))\)

Exercice 2 : Résolution d'équation

Résoudre dans \([0, 2\pi]\) l'équation : \(2\cos^2(x) - \cos(x) - 1 = 0\)

Solution :

Posons \(u = \cos(x)\). L'équation devient :

\(2u^2 - u - 1 = 0\)

Calculons le discriminant : \(\Delta = 1 + 4 \times 2 \times 1 = 9\)

\(\Delta > 0\), donc l'équation admet deux solutions :

\(u_1 = \frac{1 + 3}{4} = 1\) et \(u_2 = \frac{1 - 3}{4} = -\frac{1}{2}\)

Donc \(\cos(x) = 1\) ou \(\cos(x) = -\frac{1}{2}\)

Si \(\cos(x) = 1\), alors \(x = 0\) ou \(x = 2\pi\) dans \([0, 2\pi]\)

Si \(\cos(x) = -\frac{1}{2}\), alors \(x = \frac{2\pi}{3}\) ou \(x = \frac{4\pi}{3}\) dans \([0, 2\pi]\)

Les solutions dans \([0, 2\pi]\) sont donc : \(x = 0, \frac{2\pi}{3}, \frac{4\pi}{3}, 2\pi\)

Exercice 3 : Linéarisation

Transformer en somme l'expression : \(E = \cos(3x)\cos(5x)\)

Solution :

On utilise la formule de linéarisation : \(\cos(a)\cos(b) = \frac{1}{2}[\cos(a+b) + \cos(a-b)]\)

Avec \(a = 3x\) et \(b = 5x\), on obtient :

\(E = \cos(3x)\cos(5x) = \frac{1}{2}[\cos(3x+5x) + \cos(3x-5x)]\)

\(E = \frac{1}{2}[\cos(8x) + \cos(-2x)]\)

Comme \(\cos(-\theta) = \cos(\theta)\), on a :

\(E = \frac{1}{2}[\cos(8x) + \cos(2x)]\)
